Aplikace využívá zdrojová data ve formátu Linked Data

Kontroly ČOI.CZ (http://linked.opendata.cz/resource/dataset/coi.cz/kontroly)
Data o kontrolách provedených Českou obchodní inspekcí (ČOI) zveřejněná na webu ČOI.
Sankce ČOI.CZ (http://linked.opendata.cz/resource/dataset/coi.cz/sankce)
Data o sankcích udělených Českou obchodní inspekcí (ČOI) zveřejněná na webu ČOI.
ARES - základní informace (http://linked.opendata.cz/resource/dataset/ares/basic)
Data z registru ekonomických subjektů publikovaných webovými službami ARES
Google Maps a Google Geocoding API
Aplikace využívá mapové API společnosti Google pro zobrazovaní výsledků kontrol a Geocoding API společnosti Google pro určení GPS souřadnic, na kterých kontrola proběhla. Spolehlivost určení GPS souřadnic není 100% - vyplývá z úplnosti adres uvedených ve zdrojových datech ČOI.

Upozornění: Provozovatel VysledkyKontrol.cz neodpovídá za správnost a aktuálnost informací obsažených na jeho webových stránkách (včetně webových stránek VysledkyKontrol.cz).
